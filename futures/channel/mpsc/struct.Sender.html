pub fn poll_ready(&mut self, lw: &LocalWaker) -> Poll<Result<(), SendError>>
[src]

Polls the channel to determine if there is guaranteed capacity to send at least one item without waiting.

Return value

This method returns:

  • Ok(Async::Ready(_)) if there is sufficient capacity;
  • Ok(Async::Pending) if the channel may not have capacity, in which case the current task is queued to be notified once capacity is available;
  • Err(SendError) if the receiver has been dropped.
